Hoe gebruik ik de Praatbox?
  1. Geef je praatbox een naam. Vul dit in het eerste veld in.
  2. Geef je praatbox een locatie. Vul dit in het tweede veld in.
  3. Bedenk een willekeurig woord. Dit mag een bestaand woord zijn, maar ook een verzonnen woord. Wees dus creatief!
  4. Als je de vorige stappen hebt ondernomen, klik je op "Start de praatbox"
  5. De praatbox start op!
  6. De je kan deze praatbox nu delen op allerlei verschillende manier, het enige wat je moet doen is de link kopieren
